XMLCMS – Простая cms на Ember.js, XML, XSLT, PHP

Библиотека распространяется на условиях свободной лицензии MIT. http://opensource.org/licenses/mit-license.php 

Для работы приложения на стороне сервера необходима поддержка PHP и модуля PHP разбирающего XSLT-шаблоны. 

Возможности cms:
-CMS выполена в виде SPA (Single Page Application - Приложение одной страницы)
-Изменение дизайна сайта через XSLT-шаблоны
-Изменение содержание сайта через XML-файлы
-Встроенные модули "Страницы", "Новости"


Возможные варианты использования cms:
-Создание простого сайта-визитки, с редко меняющимся содержимым и шаблонами
-Создание Landing-page

CMS не требовательная к ресурсом сервера, так как не содержит базы данных и веб-интерфейса управления.
